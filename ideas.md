# Ideias de Design para Portfólio de Gabriel Santos

## Contexto
Gabriel Santos é um estudante de Engenharia de Computação com forte foco em pesquisa científica, machine learning e sistemas de rede. O portfólio deve comunicar competência técnica, curiosidade intelectual e capacidade de execução. O público-alvo inclui recrutadores, professores e colaboradores potenciais.

---

## Resposta 1: Minimalismo Científico com Acentos Técnicos
**Design Movement:** Bauhaus + Swiss Design + Data Visualization

**Probabilidade:** 0.08

**Core Principles:**
- Hierarquia clara através de tipografia e espaçamento, não cores
- Grid rigoroso que reflete ordem matemática
- Dados e informações como elementos visuais primários
- Foco absoluto em legibilidade e funcionalidade

**Color Philosophy:**
- Paleta monocromática: tons de cinza (charcoal, light gray, white)
- Acentos em azul profundo (oklch(0.52 0.18 250)) para destacar links, projetos e CTAs
- Verde sutil (oklch(0.65 0.12 142)) para indicar status "ativo" ou "em progresso"
- Sem gradientes; apenas cores sólidas e transições suaves

**Layout Paradigm:**
- Grid de 12 colunas com espaçamento generoso
- Seções alternadas: conteúdo à esquerda, visualizações/imagens à direita
- Navegação fixa no topo, minimalista (nome + links)
- Rodapé com links sociais e contato

**Signature Elements:**
1. **Linha de tempo visual** para experiências e projetos (vertical, com ícones técnicos)
2. **Cards de projeto** com borda esquerda colorida (cor varia por tipo: pesquisa, desenvolvimento, etc.)
3. **Badges de habilidades** com tipografia pequena e espaçamento preciso

**Interaction Philosophy:**
- Hover: mudança sutil de cor ou elevação (shadow)
- Transições suaves (0.3s) em todas as interações
- Sem animações excessivas; movimento serve propósito

**Animation:**
- Fade-in ao scroll para seções
- Slide-in suave para cards de projeto
- Underline animado em links ao hover

**Typography System:**
- Display: Playfair Display (bold, 48-56px) para título principal
- Headings: Inter (600, 24-32px) para seções
- Body: Inter (400, 16px) para texto principal
- Small: Inter (400, 14px) para metadados e badges

## Resposta 2: Estilo Cyberpunk Acadêmico
**Design Movement:** Cyberpunk + Neon + Data Art

**Probabilidade:** 0.07

**Core Principles:**
- Contraste alto e cores vibrantes para capturar atenção
- Tipografia ousada e geométrica
- Elementos de "código" e "dados" como decoração visual
- Sensação de movimento e energia

**Color Philosophy:**
- Fundo escuro: oklch(0.12 0.01 0) (quase preto)
- Primária: magenta neon (oklch(0.65 0.25 310))
- Secundária: ciano (oklch(0.68 0.22 200))
- Acentos: amarelo limão (oklch(0.85 0.18 100))
- Gradientes: magenta → ciano em backgrounds de destaque

**Layout Paradigm:**
- Assimétrico com ângulos diagonais (clip-path para seções)
- Navegação lateral esquerda com ícones grandes
- Conteúdo em colunas variáveis
- Efeito de "glitch" sutil em títulos

**Signature Elements:**
1. **Padrão de grade pixelada** como background subtil
2. **Linhas de "código" animadas** como separadores
3. **Ícones geométricos** para categorizar projetos

**Interaction Philosophy:**
- Hover: mudança de cor vibrante + glow effect
- Cliques: efeito de "ripple" ou "pulse"
- Transições rápidas (0.2s) para sensação de responsividade

**Animation:**
- Glow pulsante em elementos interativos
- Scan-line effect ao carregar seções
- Texto que "digita" ao entrar na viewport

**Typography System:**
- Display: Space Mono (bold, 48-64px) para título
- Headings: IBM Plex Mono (600, 24-32px)
- Body: IBM Plex Mono (400, 16px)
- Monospace para código e dados

## Resposta 3: Elegância Moderna com Foco em Conteúdo
**Design Movement:** Contemporary Minimalism + Content-First Design

**Probabilidade:** 0.06

**Core Principles:**
- Conteúdo é rei; design suporta, não compete
- Tipografia sofisticada como elemento principal
- Espaçamento generoso e respiração visual
- Paleta neutra com um acento cor quente

**Color Philosophy:**
- Fundo: branco puro (oklch(1 0 0))
- Texto principal: cinza muito escuro (oklch(0.25 0.01 0))
- Acentos: laranja quente (oklch(0.65 0.18 45)) para CTAs e destaques
- Cinza claro para borders e separadores (oklch(0.92 0.002 0))

**Layout Paradigm:**
- Centrado com max-width generoso (1000px)
- Seções com padding vertical amplo (80-120px)
- Tipografia em tamanhos variados para criar ritmo
- Imagens full-width entre seções

**Signature Elements:**
1. **Numeração visual** para projetos (01, 02, 03...)
2. **Citações destacadas** em itálico com acento laranja
3. **Separadores sutis** (linhas finas ou espaço branco)

**Interaction Philosophy:**
- Transições suaves e naturais
- Hover: mudança de cor do texto para laranja
- Sem efeitos disruptivos; tudo é elegante e controlado

**Animation:**
- Fade-in ao scroll (suave, 0.6s)
- Slight scale-up em hover de links
- Parallax sutil em imagens de fundo

**Typography System:**
- Display: Cormorant Garamond (700, 52-64px) para título
- Headings: Lora (600, 28-36px)
- Body: Lora (400, 18px)
- Small: Lora (400, 14px)
